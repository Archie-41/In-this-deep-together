# In-this-deep-together
Final Project for the Deep Learning course.

## Papers
We replicate the work of the following related papers: [https://arxiv.org/pdf/2205.09114](https://arxiv.org/pdf/2205.09114), [https://arxiv.org/abs/2111.04881](https://arxiv.org/pdf/2111.04881) and improve on the model architecture.

## Code
The original code of the paper can be found in this repository: [https://github.com/usnistgov/SolDet](https://github.com/usnistgov/SolDet).
Follow SolDet/README.md to setup your python environment.

## Dataset
In the folder SolDet you will find the code for the papers mentioned above.
The full dataset can be found here: https://data.nist.gov/od/id/mds2-2363.
